# Assets

Поместите сюда PNG-иконки ключевых преимуществ размером 72×72 px или больше с именами `1.png`, `2.png`, `3.png` и `4.png`. Они должны быть в фирменном красном контуре.

Для блока «Галерея меню» положите PNG/JPG-файлы с именами `menu-01.png` … `menu-10.png` (используйте оригинальные названия отправленных кадров). Размеры: от 1600 px по ширине, соотношение сторон ближе к 4:3 для лучшего отображения в карусели.
